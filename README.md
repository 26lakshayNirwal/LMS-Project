# LMS + AI Document Summarizer

## 🚀 Features

* User Authentication (Login)
* LMS Dashboard
* Document Upload
* AI-based Document Summarization (Gemini API)
* Imarticus Landing Page Clone

## 🛠 Tech Stack

* Frontend: React + Bootstrap
* Backend: Node.js + Express
* Database: MongoDB Atlas
* AI: Google Gemini API

## 🔗 Live Links

* Frontend: [(https://lms-project-1-ydsh.onrender.com)]
* Backend: [https://lms-project-tqfi.onrender.com]

## ⚙️ Setup

### Backend

cd backend
npm install
npm run dev

### Frontend

cd frontend
npm install
npm start

## 📌 Flow

Home → Apply → Login → Dashboard → Upload → Summarize

## ⚠️ Note

Fallback handling added in case of API quota issues.
